# Installing with the Codex desktop app

Open this project as a local folder in Codex and ask:

> Install this Codex usage Lock Screen widget. Run the included installer, use only the Codex executable bundled with the ChatGPT or Codex desktop app, and verify the Scriptable iCloud JSON plus LaunchAgent logs. Do not install a separate Codex CLI or expose authentication tokens.

Approve local file, iCloud Drive, and LaunchAgent permissions only when macOS asks. After installation, open **Codex Usage** once in Scriptable on the iPhone and select it as an `accessoryRectangular` Lock Screen widget.
